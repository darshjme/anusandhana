## Overview

`agent-context-tracker` implements **track context window usage across agent turns: token counting, compression triggers, context pruning strategies**

Inspired by the Vedic principle of *अनुसंधान* (Anusandhana), this library brings the ancient wisdom of structured discipline to modern LLM agent engineering.

No external dependencies. Pure Python 3.8+. Drop it in anywhere.
